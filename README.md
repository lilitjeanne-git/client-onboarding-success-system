Client Onboarding Success System

A reusable project management toolkit for client-facing teams
This repository contains a complete, ready-to-use onboarding toolkit
for Project Managers and Customer Success Managers. It covers the full
project lifecycle — from kick-off to closure — with templates that work
for any client, any industry, and any project type.
Every document is a template. For each new client, duplicate the relevant
files, replace the [placeholders], and you have a professional
project-ready toolkit in minutes.

Who This Is For

Project Managers running client delivery projects
Customer Success Managers onboarding new clients
Anyone who manages structured, multi-phase engagements with external stakeholders

How to Use This Repo
Starting a new project? Follow this order:

Fill in framework/project-charter-template.md — define the project before anything else
Set up templates/stakeholder-register.xlsx with all client and internal contacts
Open templates/communication-plan-template.xlsx and adapt it to your client
Use templates/onboarding-tracker.xlsx to track milestones and deliverables
Log risks and assumptions in templates/raid-log-template.xlsx — start before kick-off
Use templates/project-closure-checklist.xlsx to close the project cleanly

Something changing mid-project?
→ Raise a Change Request in framework/change-request-template.md before acting on it.
Using AI to draft emails, plans, or reports?
→ Paste the project charter as context first. The AI will produce output
specific to your project without you re-explaining it every time.
